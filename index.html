一步一步把网页发布到 GitHub Pages

📌 第 1 步：准备文件

  • 把网页代码保存成一个文件，命名 index.html。
  • 如果有图片，可以：
    • 放在同一个文件夹（例如 assets/），并在代码里引用 assets/xxx.jpg。
    • 或使用公开的在线图片链接。

📌 第 2 步：新建仓库

  • 登录 GitHub。
  • 右上角点击 + → New repository（如你截图所示）。
  • 仓库名随意，例如 secondhand-listings。
  • 选择 Public → 点击 Create repository。

📌 第 3 步：上传文件

  • 进入新仓库主页。
  • 点击 Add file → Upload files。
  • 拖入 index.html（和图片文件夹，如有）。
  • 滚动到底部，点击 Commit changes。

📌 第 4 步：开启 GitHub Pages

  • 进入仓库 → 顶部点击 Settings。
  • 左侧菜单选择 Pages。
  • 在 Build and deployment 区域：
    • Source → Deploy from a branch
    • Branch → main
    • Folder → / (root)
  • 点击 Save，等待 1–2 分钟。

📌 第 5 步：访问网页

  • 几分钟后会自动生成网址：
  • https://你的用户名.github.io/secondhand-listings/
  • 打开即可看到你的二手物品展示网页。

✅ 提示：以后只要更新 index.html 并上传，网页会自动刷新，无需再次设置。
